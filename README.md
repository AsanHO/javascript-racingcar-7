# javascript-racingcar-precourse

# 기능구현 리스트

- [✅] 자동차 이름 입력 
- [✅] 자동차 이름 입력 유효성 검증
- [✅] 자동차 클래스 만들기
- [✅] 리스트<자동차> 생성.
- [✅] 시도횟수 입력
- [✅] 시도 횟수 유효성 검증 (0 초과)
- [✅] 횟수만큼 반복할 함수 리스트<객체>순회 => 랜덤 함수를 돌리고, 조건 통과시 distance 속성에 "-"추가
- [✅] 리스트<객체> 순회로 현재 거리 출력 
- [✅] 최종 우승자 출력(max : distance의 길이를 통해,)
